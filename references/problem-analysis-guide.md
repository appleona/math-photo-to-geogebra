## 第一部分：函数图像类

### 1.1 一次函数 y=kx+b

#### 题型识别
- 函数解析式含有待定系数k和/或b
- 题目给出函数图像经过的点坐标
- 求参数取值范围的问题
- 两条直线位置关系问题

#### 核心概念
| 参数 | 几何意义 | 变化规律 |
|------|----------|----------|
| k（斜率） | 直线倾斜程度 | k>0时上升，k<0时下降，\|k\|越大越陡 |
| b（截距） | 与y轴交点纵坐标 | b>0交y轴正半轴，b<0交负半轴 |

#### 关键理解要点

**1. y=mx型过原点问题**
```
典型题目：y=mx(m≠0)与y=kx+b在x>a时始终有mx>kx+b，求m范围

理解要点：
- y=mx是过原点的直线
- m是斜率，控制直线绕原点旋转
- 临界情况通常发生在交点或平行时

正确做法：
✓ 用滑块控制m（范围建议0~2或-2~2）
✓ 动态展示直线旋转过程
✓ 标出临界点和边界线
```

**2. 参数范围问题**
```
典型题目：当m满足什么条件时，直线y=mx+2与线段AB有交点

理解要点：
- 需要考虑边界情况（过端点）
- 可能需要分类讨论
- 注意开区间和闭区间的区别

正确做法：
✓ 用滑块控制参数
✓ 展示"有交点"和"无交点"两种状态
✓ 标出临界值
```

#### 常见陷阱

| 陷阱 | 错误做法 | 正确做法 |
|------|----------|----------|
| 静态展示 | 画出m=0.5和m=1两条线 | 用滑块控制m动态变化 |
| 忽略原点 | 随意放置y=mx | 确保过原点O(0,0) |
| 范围不完整 | 只考虑m>0 | 同时考虑m<0的情况 |
| 端点遗漏 | 忽略等号是否成立 | 明确临界状态是否满足 |

#### GeoGebra实现

```xml
<!-- 滑块控制斜率m，范围0到1.5 -->
<element type="numeric" label="m">
    <value val="0.5"/>
    <slider min="0" max="1.5" step="0.01" type="number"/>
    <show object="true" label="true"/>
</element>

<!-- 动态直线 y = m*x -->
<element type="function" label="g_dynamic">
    <expression label="g_dynamic" exp="m*x" type="function"/>
    <show object="true" label="true"/>
    <objColor r="0" g="128" b="0" alpha="0"/>
    <lineStyle thickness="3" type="0"/>
</element>

<!-- 固定参考直线 -->
<element type="function" label="f">
    <expression label="f" exp="0.5*x - 1" type="function"/>
    <show object="true" label="true"/>
    <objColor r="0" g="0" b="255" alpha="0"/>
    <lineStyle thickness="3" type="0"/>
</element>
```

### 1.2 二次函数 y=ax²+bx+c

#### 题型识别
- 抛物线相关问题
- 与x轴交点、顶点问题
- 二次函数与几何图形综合
- 动点与二次函数结合

#### 核心概念

| 参数 | 作用 | 图像特征 |
|------|------|----------|
| a | 开口方向和张口大小 | a>0向上，a<0向下，\|a\|越大开口越小 |
| -b/(2a) | 对称轴 | 顶点横坐标 |
| (4ac-b²)/(4a) | 顶点纵坐标 | 最值 |
| Δ=b²-4ac | 判别式 | Δ>0两交点，Δ=0一交点，Δ<0无交点 |

#### 关键理解要点

**1. 顶点问题**
```
理解要点：
- 顶点坐标必须准确计算
- 顶点可能是最值点
- 需要考虑a的正负确定最大/最小值

正确做法：
✓ 使用公式计算顶点坐标
✓ 在GeoGebra中标记顶点
✓ 展示对称轴
```

**2. 交点问题**
```
典型题目：抛物线与x轴交于A、B两点，求三角形ABC面积

理解要点：
- 先求交点坐标（解方程）
- 再计算相关几何量
- 注意Δ的符号判断

正确做法：
✓ 使用Root工具或解方程求交点
✓ 动态展示参数变化时交点的变化
```

#### 常见陷阱

| 陷阱 | 说明 | 避免方法 |
|------|------|----------|
| 顶点计算错误 | 忘记除以4a或符号错误 | 仔细核对公式 |
| 忽略a的符号 | 最值判断错误 | 明确a>0是最小值，a<0是最大值 |
| 交点遗漏 | 只找一个交点 | Δ判断，确保找到所有交点 |

### 1.3 反比例函数 y=k/x

#### 题型识别
- 双曲线相关问题
- k的几何意义问题
- 与一次函数交点问题

#### 核心概念

| k值 | 图像位置 | 单调性 |
|------|----------|--------|
| k>0 | 第一、三象限 | 每支内y随x增大而减小 |
| k<0 | 第二、四象限 | 每支内y随x增大而增大 |

#### 关键理解要点

**1. k的几何意义**
```
理解要点：
- 图像上任意一点P(x,y)向坐标轴作垂线
- 与原点围成的矩形面积为|k|
- 三角形面积为|k|/2

正确做法：
✓ 在GeoGebra中标注这个矩形/三角形
✓ 动态展示k变化时面积的变化
```

**2. 双曲线的两支**
```
理解要点：
- 反比例函数图像有两支
- 不能连成一个整体
- x≠0，图像无限接近但永不到达坐标轴

正确做法：
✓ 画出两支完整的曲线
✓ 注意渐近线的表现
```

#### GeoGebra实现

```xml
<!-- 滑块控制k值 -->
<element type="numeric" label="k">
    <value val="6"/>
    <slider min="-10" max="10" step="0.5" type="number"/>
</element>

<!-- 反比例函数 -->
<element type="function" label="f">
    <expression label="f" exp="k/x" type="function"/>
</element>
```

## 第二部分：几何变换类

### 2.1 轴对称（翻折/折叠）

#### 题型识别
- 题目中出现"折叠"、"翻折"、"对称"
- 矩形/三角形沿某条线折叠
- 求折叠后对应点位置或线段长度

#### 核心概念
- 对称轴是对应点连线的垂直平分线
- 折叠前后图形全等
- 折痕就是对称轴

#### 关键理解要点

**1. 折叠问题的本质**
```
典型题目：矩形ABCD沿EF折叠，使C落在AD边上的C'处

理解要点：
- 折叠=轴对称变换
- 对称轴是折痕EF
- 对应点到对称轴距离相等
- 对应线段相等，对应角相等

正确做法：
✓ 确定对称轴（折痕）
✓ 找到对应点关系
✓ 利用全等性质建立方程
```

**2. 动态折叠**
```
理解要点：
- 折叠位置可以变化
- 可以用动点控制折痕位置
- 观察折叠后图形的运动轨迹

正确做法：
✓ 用滑块或动点控制折叠位置
✓ 使用Mirror工具创建对称图形
✓ 展示不同折叠位置的结果
```

#### GeoGebra实现

```xml
<!-- 创建对称图形 -->
<command name="Mirror">
    <input a0="pointC"/>  <!-- 要对称的点 -->
    <input a1="foldLine"/> <!-- 对称轴（折痕） -->
    <output a0="C_prime"/>
</command>

<!-- 动点控制折叠位置 -->
<command name="Point">
    <input a0="sideAB"/>  <!-- 折痕端点在AB上移动 -->
    <output a0="E"/>
</command>
```

#### 常见陷阱

| 陷阱 | 说明 | 避免方法 |
|------|------|----------|
| 对称轴确定错误 | 折痕位置搞错 | 仔细审题，确认折叠方式 |
| 对应点找错 | C'的位置不正确 | 利用垂直平分线性质确定 |
| 忽略全等关系 | 忘记折叠前后相等 | 明确所有对应边、对应角 |

### 2.2 旋转

#### 题型识别
- 题目中出现"旋转"、"转动"
- 等边三角形、正方形中的旋转
- 求旋转后的坐标或位置

#### 核心概念
- 旋转中心：固定不动的点
- 旋转角：旋转的角度大小
- 旋转方向：顺时针或逆时针
- 对应点到旋转中心距离相等
- 对应点与旋转中心连线夹角等于旋转角

#### 关键理解要点

**1. 旋转三要素**
```
理解要点：
- 旋转中心必须明确
- 旋转角度必须准确
- 旋转方向不能搞反

正确做法：
✓ 标出旋转中心
✓ 用滑块控制旋转角度
✓ 明确标注顺时针/逆时针
```

**2. 特殊旋转**
```
等边三角形内：旋转60°重合
正方形内：旋转90°重合
圆内：旋转任意角度

理解要点：
- 利用特殊角的性质
- 旋转后可能产生新的特殊图形

正确做法：
✓ 展示旋转过程中的关键位置
✓ 标注旋转后形成的特殊角
```

#### GeoGebra实现

```xml
<!-- 滑块控制旋转角度 -->
<element type="numeric" label="angle">
    <value val="30"/>
    <slider min="0" max="360" step="1" type="angle"/>
</element>

<!-- 旋转命令 -->
<command name="Rotate">
    <input a0="pointP"/>       <!-- 要旋转的点 -->
    <input a1="angle"/>        <!-- 旋转角度 -->
    <input a2="centerO"/>      <!-- 旋转中心 -->
    <output a0="P_rotated"/>
</command>
```

#### 常见陷阱

| 陷阱 | 说明 | 避免方法 |
|------|------|----------|
| 方向错误 | 顺时针逆时针搞反 | 明确标注方向，用角度正负表示 |
| 中心错误 | 旋转中心选错 | 仔细审题，确认哪个点不动 |
| 角度计算错误 | 旋转角不等于对应角 | 注意区分旋转角和图形内角 |

#### 题型识别
- 题目中出现"点P在AB上移动"
- 求最值问题
- 求满足某条件的点的位置

#### 核心理解
```
关键：点必须约束在线段上，不是自由点

正确做法：
✓ 使用Point工具约束到线段
✓ 动点拖动时始终在线段上
✓ 相关几何元素自动跟随变化
```

#### GeoGebra实现

```xml
<!-- 先创建线段 -->
<command name="Segment">
    <input a0="A" a1="B"/>
    <output a0="segAB"/>
</command>

<!-- 动点P约束在线段AB上 -->
<command name="Point">
    <input a0="segAB"/>
    <output a0="P"/>
</command>
<element type="point" label="P">
    <show object="true" label="true"/>
    <objColor r="255" g="0" b="0" alpha="0"/>
    <pointSize val="5"/>
</element>
```

## 第四部分：三角形类

### 4.1 全等三角形

#### 判定方法（按优先级）
1. **SAS** - 两边及其夹角
2. **ASA** - 两角及其夹边
3. **AAS** - 两角及其中一角的对边
4. **SSS** - 三边
5. **HL** - 直角三角形斜边直角边

#### 理解要点
```
⚠️ 注意：SSA不能判定全等！

正确做法：
✓ 明确对应顶点顺序
✓ 对应边、对应角准确匹配
✓ 利用全等转移线段/角度关系
```

---

### 4.2 等腰三角形

#### 核心性质
- 等边对等角，等角对等边
- 三线合一：顶角平分线、底边中线、底边高

#### 分类讨论提醒
```
⚠️ 等腰三角形问题必须分类讨论：

情况1：已知一角，可能是顶角或底角
- 若顶角=40°，则底角=(180°-40°)/2=70°
- 若底角=40°，则顶角=180°-2×40°=100°

情况2：已知两边，可能是腰或底
- 若3和4是两边，第三边可能是3（腰）或4（腰）
- 但必须满足两边之和大于第三边

正确做法：
✓ 明确写出所有可能情况
✓ 检验是否满足三角形条件
✓ 在GeoGebra中展示不同情况
```

---

### 4.3 相似三角形

#### 判定方法
1. **AA** - 两角对应相等
2. **SAS** - 两边成比例且夹角相等
3. **SSS** - 三边成比例

#### 常见模型

| 模型 | 图形特征 | 相似关系 |
|------|----------|----------|
| A字型 | 三角形内一条平行于底边的线段 | △ADE∽△ABC |
| 八字型 | 两条相交直线被平行线截取 | △AOB∽△COD |
| 射影定理 | 直角三角形斜边上的高 | 三个三角形两两相似 |

#### 理解要点
```
⚠️ 注意：面积比 = 相似比的平方

正确做法：
✓ 准确找出对应顶点
✓ 相似比要对应边相比
✓ 面积计算记得平方
```

## 第五部分：四边形类

### 5.1 平行四边形判定与性质

#### 判定方法（五种）
1. 两组对边分别平行
2. 两组对边分别相等
3. 一组对边平行且相等
4. 两组对角分别相等
5. 对角线互相平分

#### 理解要点
```
平行四边形性质应用：
- 对边平行→内错角相等、同旁内角互补
- 对角线互相平分→中点问题
- 对角相等、邻角互补→角度计算
```

---

### 5.2 特殊平行四边形

| 图形 | 特有性质 | 判定方法 |
|------|----------|----------|
| 矩形 | 四个直角、对角线相等 | 平行四边形+一个直角/对角线相等 |
| 菱形 | 四边相等、对角线垂直平分对角 | 平行四边形+邻边相等/对角线垂直 |
| 正方形 | 兼具矩形和菱形所有性质 | 既是矩形又是菱形 |

#### 理解要点
```
正方形问题的关键：
- 等腰直角三角形（对角线分成的）
- 45°角的利用
- 对角线相等且垂直平分

菱形问题的关键：
- 对角线互相垂直
- 面积=对角线乘积/2
- 勾股定理应用
```

## 第六部分：圆类

### 6.1 圆的基本性质

#### 重要定理
- **垂径定理**：垂直于弦的直径平分弦且平分弦所对的两条弧
- **圆周角定理**：同弧所对圆周角等于圆心角的一半
- **圆内接四边形**：对角互补

### 6.2 与圆有关的位置关系

| 位置 | 条件 | 交点个数 |
|------|------|----------|
| 点与圆 | d<r圆内，d=r圆上，d>r圆外 | - |
| 直线与圆 | d<r相交，d=r相切，d>r相离 | 2,1,0 |
| 圆与圆 | 比较圆心距与半径和/差 | 2,1,0 |

#### 切线问题要点
```
切线的性质：
- 切线垂直于过切点的半径
- 圆心到切线距离等于半径
- 切线长定理

正确做法：
✓ 连接圆心和切点
✓ 利用垂直关系建立方程
```
