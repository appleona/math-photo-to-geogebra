## 它能解决什么问题？

### ✅ 动点问题 —— 再也不用"脑补"

> **原题**：点P在BC上移动，求PA+PB的最小值
>
> **传统**：孩子盯着图发呆，想象P怎么动
>
> **我们**：生成一个文件，孩子**拖动红色点P**，实时看到PA+PB的长度变化，找到最小值位置

### ✅ 函数参数 —— 一眼看懂"为什么"

> **原题**：y=mx在x>-2时始终大于y=0.5x-1，求m范围
>
> **传统**：列不等式、解方程，机械计算
>
> **我们**：生成一个带**滑块**的课件，孩子拖动滑块改变m，亲眼看到直线怎么转、什么时候"超标"

### ✅ 几何变换 —— 折叠旋转看得见

> **原题**：矩形沿EF折叠，使C落在AD边上
>
> **传统**：折纸？画图？都麻烦
>
> **我们**：生成可拖动的折叠动画，孩子拖动折痕位置，实时看到折叠后的图形

### ✅ 坐标系综合 —— 数形结合真落地

> **原题**：抛物线与直线交于A、B，求三角形面积
>
> **传统**：算坐标、套公式、易出错
>
> **我们**：图形上直接显示交点坐标、面积数值，拖动参数实时变化

## 使用方法（超简单）

### 第一步：准备题目

拍一张数学题照片，或者把题目文字准备好。

### 第二步：运行工具

```bash
# 把题目转换成GeoGebra XML
python scripts/pack_ggb.py output.ggb -i input.xml
```

### 第三步：打开课件

**方式一（推荐）：网页版 —— 无需安装**

1. 打开 [https://www.geogebra.org/classic](https://www.geogebra.org/classic)
2. 点击左上角 **📁 打开** → **浏览**
3. 选择生成的 `.ggb` 文件，即可在浏览器中直接使用

> ✅ 零安装，手机/平板/电脑都能用
> ✅ 打开即用，孩子随时可学

**方式二：桌面版 —— 功能更完整**

下载安装 **GeoGebra Classic 6**（免费）：
[https://www.geogebra.org/download](https://www.geogebra.org/download)

双击 `.ggb` 文件即可打开。

### 第四步：动手探索

- **拖动红点** —— 动点随你移
- **拖动滑块** —— 参数随你调
- **观察变化** —— 规律自己发现

## 真实案例：一道题的两种学法

### 📖 传统学法

> 题目：菱形ABCD边长8，∠ABC=120°，E是AB中点，M在AC上动，求BM+EM最小值。
>
> 孩子：画图画半天，辅助线想不到，看答案才知道要"对称转化"。
>
> 结果：似懂非懂，换道题又不会。

### 🎯 我们的学法

> 生成课件后，孩子看到：
> - 一个**红色动点M**在AC上
> - 拖动M，屏幕实时显示 **BM+EM = ?**
> - 发现：当M走到某个位置时，和最小！
> - 再观察：原来这个位置是DE和AC的交点！
> - 恍然大悟：B和D对称，所以BM=DM，最短就是线段DE！
>
> 结果：**自己发现规律**，印象深刻，换道题也能举一反三。

## 我们覆盖了哪些内容？

**人教版初中数学（七到九年级）全部数形结合内容：**

- **三角形** —— 全等、相似、等腰、直角三角形动态演示
- **四边形** —— 平行四边形、矩形、菱形、正方形的性质与变换
- **函数** —— 一次函数、二次函数、反比例函数的图像与参数
- **圆** —— 圆的性质、切线、位置关系
- **几何变换** —— 平移、旋转、轴对称（折叠）
- **动点问题** —— 最值、存在性、轨迹
